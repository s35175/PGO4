# pgo4 - Cafe Console Application
A simple console application designed for a small cafe to register menu items, customer data, and individual orders.

## Class List
* **MenuItem**: Represents a single product sold in the cafe.
* **CafeCustomer**: Represents a customer placing an order.
* **Order**: Represents a single customer order containing multiple MenuItems.
* **Main**: Contains the test scenario to run the application.
